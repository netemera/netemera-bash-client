netemera-bash-client
====================

Simple netemera API bash client written in Bash.

### Dependencies

- curl
- sed
- Bash >4.0
- coreutils

### Setup

Create file named `netemera-as-api.conf` in `/etc` or in `~/.config` directory containing:

```
CLIENT_ID=...
CLIENT_SECRET=...
AUTHORIZATION_HOST=...
APPLICATION_HOST=...

```
